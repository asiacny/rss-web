# rss-web
> 一个简洁的web版的rss阅读器，vue+vue-router+koa2+sqlite+redis

功能实现:
- 用户登录注册
- 分类管理
- rss管理
- 文章进行了redis 1小时缓存，不过可以通过右下角的刷新按钮进行手动刷新，刷新进行了频率限制(两次刷新至少间隔10分钟)
- 后端接口采用cors实现跨域
